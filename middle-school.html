Middle School Camp

In this version of camp, you will teach a robot how to react to buttons, objects, and lines using simple code. You do not need to build the robot from scratch. It is already ready to drive, grab, and test ideas.

Camp goals

  • Understand that sensors help robots notice what is happening.
  • Use simple if/then logic to make the robot react.
  • Practice testing, fixing, and improving code.
  • Complete fun missions with a claw robot.

Sensor overview

  • Pressure sensor: a push button the robot can feel.
  • Range sensor: helps the robot tell when something is close.
  • Optical sensor: helps the robot notice a dark line or change in surface.

Mission 1: Pressure Sensor Start Button

Press the back sensor to start the robot. When the button is pressed, the robot drives forward for a short distance.

Big idea: a pressed button is input, and moving is output.

Mission 2: Range Sensor Claw Grabber

Drive toward an object. When the front range sensor sees it nearby, stop and close the claw.

Big idea: the robot can react before it bumps into something.

Mission 3: Optical Sensor Surface or Line Detector

Let the robot drive until the bottom optical sensor notices a dark line or different surface, then stop.

Big idea: robots can follow rules based on what the floor looks like.

Final Challenge: Robot Rescue Mission

Pick up a small object and deliver it to a safe zone. Use sensor clues to know when to start, grab, and stop.

Try combining more than one mission idea into one program.

Reflection questions

  • Which sensor was the easiest to understand? Why?
  • What happened when the robot did not do what you expected?
  • How did the sensors help the robot make better choices?
  • Where have you seen sensors used in real life?
