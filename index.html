What this camp is about

Students learn that sensors are how a robot notices change. A pressure sensor can act like a button, a distance sensor can tell when an object is close, and an optical sensor can help a robot detect lines, colors, or changes in surface brightness.

Both camp versions fit into a 2-hour session. The middle school path is more guided and block-based. The high school path goes further with thresholds, variables, loops, debugging, and autonomous choices.

What is a sensor?

A sensor is a tool that helps a robot gather information from the world. That information becomes input. The robot uses code to make a decision, then performs an output such as driving, stopping, or closing the claw.

Input

Sensor reads a press, distance, or surface change

Logic

Code checks rules like if/then, thresholds, or loops

Output

Robot moves, stops, turns, or grabs an object

Why claws and sensors work well together

The claw gives students a visible job for the robot to do: notice something, move toward it, and grab it. That makes each sensor feel useful instead of abstract. Students quickly see how sensing leads to action.

Optional smart home connection

Campers can also see that sensor logic is not only for robots. With Raspberry Pi, Home Assistant, Aqara Zigbee motion sensors, door sensors, smart bulbs, and smart switches, you can show how a home can sense movement or an open door and trigger lights or other automations.
