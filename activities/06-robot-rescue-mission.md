# Mission 6: Robot Rescue Mission

- Time estimate: 25 minutes

## Goal

Combine sensor ideas to complete a short rescue-style autonomous mission.

## Story Theme

The robot must find a target, pick up an object, and deliver it to a safe zone without crashing into anything.

## Mission Requirements

- Start the robot safely
- Find or approach an object
- Grab the object with the claw
- Move it to a target or safe zone
- Stop in the correct place

## Student-Facing Rules

- Work from the mission goal, not from a fixed script
- Test one change at a time
- Keep the code readable with comments
- Use the robot's sensors to make decisions

## Middle School Version

- Give students a guided rescue path with fewer decisions
- Let them use one sensor to start, one sensor to stop, and the claw to grab
- Focus on completing the mission rather than writing a complicated program

## High School Version

- Design a more autonomous routine using at least two sensors
- Tune the behavior with thresholds and variables
- Use debugging information to explain why the robot made each choice

## Required High School Concepts

- At least two sensors
- One loop
- One if/else decision
- One variable
- Comments explaining the logic

## Optional Emergency Stop Using the Pressure Sensor

- Press the back sensor to stop the robot if the mission needs a reset
- Use this as a safety or teacher intervention input

## Reflection Questions

- Which sensor mattered most in the rescue mission?
- What was the hardest decision for the robot to make?
- How did you debug the final routine?
- What would you improve if you had five more minutes?
