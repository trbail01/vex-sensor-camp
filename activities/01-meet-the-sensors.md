# Mission 1: Meet the Robot's Senses

- Time estimate: 10-15 minutes

## Goal

Help students locate each sensor on the robot and understand what it does before they start coding.

## Sensors Used

- Pressure sensor on the back
- Optical sensor on the bottom
- Range sensor on the front near the claw

## Setup

- Place the robot where every team can see it
- Make sure the claw is open so the front sensor is easy to point out
- Open the programming software if you want to show live values

## Student Task

1. Find each sensor on the robot.
2. Say what kind of information each sensor collects.
3. Test one sensor at a time and describe what changes when it is activated.
4. Match each sensor to a real-world example.

## Middle School Version

- Use simple language: press, see, close, and stop.
- Ask students to point to each sensor and say what it notices.
- Keep the focus on understanding, not perfect technical terms.

## High School Version

- Ask students to view live sensor values if available.
- Have them describe whether each sensor acts like a switch or a measuring tool.
- Ask which sensor would be most useful for autonomous decisions and why.

## Discussion Questions

- Which sensor was easiest to understand?
- Which sensor seems most useful for driving?
- Which sensor seems most useful for the claw?
- Where have you seen a similar sensor in daily life?

## Instructor Notes

- This mission should feel quick and low-pressure.
- Use it to establish the camp vocabulary: sensor, input, output, logic.
- If students are unsure, show one example and let them restate it in their own words.
