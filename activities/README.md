The camp is organized as missions so each task has a clear goal, a small win, and a reason to use sensors.

## Recommended Order

1. Start with the robot tour and sensor naming.
2. Use the pressure sensor for an easy first success.
3. Move to distance-based stopping.
4. Combine driving with the claw.
5. End line detection with the optical sensor.
6. Finish with the rescue challenge.

## Middle School vs High School Depth

- Middle school: keep each mission guided and focused on simple if/then behavior.
- High school: use the same missions, but ask for thresholds, loops, variables, and debugging notes.
