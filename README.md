# VEX Sensor Camp

Hands-on VEX robotics camp materials for teaching middle school and high school students how robots use sensors to sense, decide, and act.

## What this project is

This repository contains a GitHub Pages-ready starter website for a summer robotics program called **VEX Sensor Camp**. The site includes:

- A homepage
- Middle school camp overview
- High school camp overview
- Instructor guide
- Resources page

## View the site locally

Because the site uses plain HTML and CSS, you can preview it with any simple web server.

### Option 1: Python

```bash
cd /home/runner/work/vex-sensor-camp/vex-sensor-camp
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

### Option 2: Open the files directly

You can also open `index.html` in a browser, although using a local server more closely matches GitHub Pages.

## Publish with GitHub Pages

1. Push the repository to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the branch you want to publish and the `/ (root)` folder.
6. Save the settings and wait for the Pages deployment to finish.

GitHub will then publish the site using the HTML files in the repository root.
