Use this page for vocabulary, sample logic, and examples that connect robot sensors to everyday technology.

Vocabulary

Sensor
A device that collects information from the world.
Input
The information a robot or system receives.
Output
The action a robot or system performs.
Condition
A rule checked by the program, such as whether a button is pressed.
Threshold
A chosen value that tells the program when to react.
Loop
Code that repeats.
Automation
A system that reacts automatically based on rules.
Debugging
Finding and fixing problems in a program.

Basic pseudocode examples

when started
  if pressure sensor is pressed
    drive forward

forever
  if distance < threshold
    stop
    close claw

forever
  if optical sensor detects dark
    stop

Example logic

  • If pressure sensor is pressed, start.
  • If distance is less than a value, stop and close claw.
  • If optical sensor detects dark surface, stop.

Real-world sensor examples

  • Robot vacuum avoiding stairs or walls
  • Automatic doors opening when someone approaches
  • Smartphones adjusting screen brightness
  • Cars using parking sensors and backup cameras
  • Smart homes turning lights on with motion sensors
  • Security systems noticing doors, windows, or movement
