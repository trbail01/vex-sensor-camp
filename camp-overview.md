# Camp Overview

## Purpose

VEX Sensor Camp introduces students to the idea that robots become more useful when they can sense their environment. Instead of just driving in a straight line, the robot can notice buttons, objects, lines, and surface changes, then react in a purposeful way.

## Learning Goals

- Understand the difference between input, logic, and output
- Recognize that sensors let robots collect information
- Write simple code that reacts to a sensor
- Practice testing, debugging, and improving a robot program
- Connect robot sensors to real-world automation systems

## Big Idea

Robots become more useful when they can sense their environment. A sensor turns the world into information the robot can use.

## What Students Will Do

- Explore the robot and identify where each sensor is located
- Complete short missions that use one sensor at a time
- Combine sensor ideas into a final challenge
- Talk about how the same ideas show up in homes, phones, cars, and security systems

## Sensor List

| Sensor | What it does | Camp use |
| --- | --- | --- |
| Pressure sensor | Detects when something presses against it | Start the robot or act like an emergency stop |
| Optical sensor | Detects color, brightness, or surface changes | Find a line, stop at a zone, or detect a floor change |
| Range sensor | Measures how far away an object is | Stop before hitting a wall or grab an object |
| Claw mechanism | Not a sensor, but an output the robot uses | Pick up and move objects |

## Recommended Group Size

- Best option: 2 students per robot
- If needed: 3 students per robot with clear roles
- Helpful roles: driver, coder, tester

## Suggested Pacing

- 10-15 minutes: welcome and robot overview
- 15-20 minutes: simple sensor demo and first mission
- 20 minutes each: guided practice missions
- 20-25 minutes: final rescue challenge
- 5-10 minutes: reflection and share-out

## Real-World Connection

The same pattern appears in everyday technology:

- A robot vacuum senses walls or stairs
- Automatic doors sense people approaching
- Smartphones adjust screens based on light or motion
- Cars use parking sensors and driver-assist systems
- Smart homes use motion, door, and light sensors to trigger automations
